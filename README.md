# API Integration

## 📱 TechnoHacks App Development Internship

### 📌 Task 3: API Integration

This Flutter application demonstrates how to fetch and display data from a REST API.

## 🚀 Features

- Fetches data from a public REST API
- Displays posts in a scrollable list
- Loading indicator while fetching data
- Simple Material Design UI

## 🛠 Technologies Used

- Flutter
- Dart
- HTTP Package
- JSON API

## API Used

https://jsonplaceholder.typicode.com/posts

## ▶️ Run the Project

```bash
flutter pub get
flutter run
```
